Polymorphism

Polymorphism is a Java OOP concept that lets programmers use the same Object to behave differently in different contexts. There are different forms of Polymorphism in Java, the main ones are method overloading and method overriding

Method Overloading

When a class has multiple methods with the same name, but different parameters, it is known as Method Overloading. Method Overloading allows the readability of the program to increase as one

            
            // Recursive method used to to return the gcd of a and b
            static int gcd(int a, int b)
            {
                if (b == 0)
                    return a;
                return gcd(b, a % b);
            }
